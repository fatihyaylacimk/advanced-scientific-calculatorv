# Advanced Scientific Calculator

A modern, responsive, and feature-rich *scientific calculator web application* built with *HTML, CSS, and JavaScript*.  
This project combines a clean mobile-style interface with advanced mathematical functions, persistent history, memory controls, and an interactive user experience for both desktop and mobile devices.

---

## Preview

Advanced Scientific Calculator is designed to provide both *basic* and *scientific* calculations in a single interface.  
It includes support for standard arithmetic operations, trigonometric functions, logarithmic functions, factorials, power operations, angle mode switching, and local history storage.

---

## Features

- Responsive mobile-friendly design
- Modern calculator UI
- Dark / Light mode
- Standard arithmetic operations
- Scientific functions:
  - sin, cos, tan
  - asin, acos, atan
  - log, ln
  - sqrt
  - x^y
  - factorial
  - abs
  - pi and e
- DEG / RAD angle mode
- Calculation history
- Local storage support for saved history
- Memory functions:
  - MC
  - MR
  - M+
  - M-
- Keyboard support
- Copy result button
- ANS feature for reusing the last result
- Touch-friendly button interactions
- Smooth mobile swipe-style visual feedback
